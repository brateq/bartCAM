== Welcome to bartCAM

bartCAM is a VSaaS (Video Surveillance as a Service) written with a Ruby on Rails, and packages Motion and AVconv. 

== Getting started

First, install new versions of Motion and AVconv packages. 
To run bartCAM you need at least Ruby 2.0 and Rails 4.0.

After that run:
        rake db:setup
        
And everything should work :)
